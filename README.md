# Gantt Chart Maker (Aganty)

A sleek, dynamic, and fully responsive Gantt Chart application built to modernize project mapping. 

Live Demo: [https://aganty.edgeone.app/](https://aganty.edgeone.app/)

## Key Features

- **Dynamic Timelines**: Visually map out complex projects by adjusting durations, start weeks, and colors instantly. Effortlessly reorder tasks using intuitive drag-and-drop mechanics.
- **Light & Dark Themes**: Fully integrated theme toggling. Seamlessly switch between a visually distinct Dark Mode and a beautifully curated Light Beige mode to match your environment.
- **Local Database Linking**: Rest easy knowing your data isn't trapped in local storage. Export your entire timeline configurations (along with hidden notes and customized labels) specifically as lightweight `.db` backup files. Later, immediately import them to flawlessly resume where you left off!
- **Data-Rich Notes**: Append custom rich-text notes inside individual task models. These dynamically show up underneath your tasks inside the sidebar and visually anchor directly onto the timeline's colored task-blocks for quick-access editing!
- **PNG Extraction**: Easily generate and download transparently mapped, high-resolution PNG snapshots of your active timelines in one click for pitching or presentations.
